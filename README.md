# Simple Hello World

A colorful and fun Node.js "Hello World" application with ASCII art and system information display.

## Features

- Colorful ASCII art "Hello World" text
- System information display
- Simple and clean code structure
- Ready for expansion

## Installation

```bash
git clone <repository-url>
cd simple-hello-world
npm install
```

## Usage

```bash
npm start
```

or

```bash
node app.js
```

## Dependencies

- **colors** (^1.4.0) - Add colors to console output
- **figlet** (^1.5.2) - Create ASCII art text
